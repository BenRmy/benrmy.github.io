四、其他

不想使用hexo默认的主题，可以修改主题：

  1. 在官网挑选一个主题
  2. 然后在github下载或clone该主题项目
  3. 放到项目的themes文件夹下
  4. 修改_config.yml设置主题以及安装相关的依赖等
  5. 修改_config.yml文件，将theme:后的主题名修改为刚才下载的主题

    注意：主题名就是下载的主题的文件夹名，可能需要修改
